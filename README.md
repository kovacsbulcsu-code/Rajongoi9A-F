#  Minecraft Fan Page

Ez egy **Minecraft rajongói weboldal**, amely a játék világát, érdekességeit és hasznos információit mutatja be.
A projekt tanulási célból készül (HTML / CSS / JavaScript gyakorlás).

---

##  A projekt célja

- Webfejlesztési alapok gyakorlása
- GitHub + Git workflow használata csapatmunkában
- Egy egyszerű, átlátható fanoldal létrehozása

---

##  Tervezett funkciók

-  Főoldal (bemutatkozás)
-  Minecraft világok és biomok
-  Mobok és bossok leírása
-  Crafting receptek
-  Képgaléria
-  Reszponzív design (mobilon is jól nézzen ki)

---

##  Használt technológiák

- HTML5
- CSS3
- JavaScript
- Git & GitHub
- Visual Studio Code

---

##  Fejlesztési folyamat

- A `main` branch védett
- Minden módosítás külön branch-en készül
- Változtatások Pull Request-en keresztül kerülnek be
